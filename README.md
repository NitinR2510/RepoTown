Add gitcity_storybook.html — a self-contained interactive demo that visualizes a GitHub user's repositories as a cityscape. Includes styles, a canvas renderer for sky/ground/hills/roads/towers and animated characters, UI chrome (welcome, topbar, night toggle, tooltip, loading/error), GitHub API fetching (with optional token), repo-to-language grouping and house generation, and a modal room view that lists repos with a shelf animation. Provides client-side error/loading handling and sample HOUSES data for offline preview.
